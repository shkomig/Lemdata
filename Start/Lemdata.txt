## 📚 תכנון מערכת Lemdata – מערכת חינוכית חכמה

### 🎯 חזון המערכת
**Lemdata** היא מערכת חינוכית מודרנית, מבוססת ענן, המאפשרת לתלמידים, מורים והורים גישה מותאמת אישית לתכנים לימודיים, תמיכה חכמה בזמן אמת, ניתוח התקדמות, ושילוב עם מערכות קיימות כמו Coursera ומשרד החינוך.

---

### 🧩 רכיבי מערכת Lemdata

#### 1. ממשק משתמש (Frontend)
- אפליקציה רספונסיבית (Web + Mobile)
- ממשק מותאם לילדים, מורים והורים
- תמיכה בשפות שונות (עברית, אנגלית וכו’)
- אפשרות צילום והעלאת תמונות/מסמכים

#### 2. שרת מרכזי (Backend)
- Node.js / Python Flask / FastAPI
- ניהול משתמשים והרשאות
- ניהול קורסים ותכנים
- API Gateway לניהול קריאות חיצוניות

#### 3. בסיס נתונים
- PostgreSQL / MongoDB
- שמירת פרטי משתמשים, קורסים, תכנים, סטטיסטיקות

#### 4. אינטגרציות API
| שירות | תפקיד |
|-------|--------|
| Coursera API | משיכת קורסים קיימים |
| Google Cloud Vision / Azure Vision | זיהוי תמונה וסריקת מסמכים |
| OpenAI / Anthropic / Copilot | תמיכה חכמה, מענה לשאלות, סיכומים |
| משרד החינוך API (אם קיים) | סנכרון תכנים רשמיים

---

### 🤖 סוכנים חכמים ב-Lemdata
- GPT: מענה לשאלות, סיכום תכנים, תרגום
- Claude: ניתוח רגשי, תמיכה רכה
- Copilot: עזרה בקוד, תרגול טכנולוגי
- שילוב לפי הקשר: כל סוכן מופעל בהתאם לצורך המשתמש
